# Awakening Factors / Faculties / Strengths Cluster Map

This document governs the repository's handling of the
`bojjhaṅga / indriya / bala` interface.
It is not a general essay. It exists to stop recurring practice-framework
language from collapsing into one vague English blur.

## Core Framework Heads

| Term | Default | Functional role | Do not collapse into |
| --- | --- | --- | --- |
| `bojjhaṅga` | `awakening factor` | seven-member development and balancing framework | `indriya`, `bala`, static metaphysical quality |
| `indriya` | `faculty` | governing capacity within practice | `bala`, sense organ, generic power |
| `bala` | `strength` | stabilized strength that does not easily collapse | `indriya`, force, mystical power |

## Member Defaults

### Shared Faculty / Strength Member Set

| Term | Default | Functional role | Do not collapse into |
| --- | --- | --- | --- |
| `saddhā` | `confidence` | confidence that supports and steadies practice | belief, faith |
| `viriya` | `energy` | energetic application in practice | `sammā-vāyāma`, effort as the default |
| `sati` | `remembering` | recollective keeping-in-mind and gatekeeping function | mindfulness, bare awareness |
| `samādhi` | `mental composure` | broader composure family within practice | `sammā-samādhi`, concentration |
| `paññā` | `discernment` | cultivated capacity of discernment | knowledge, generic wisdom-talk |

### Awakening-Factor Member Set

| Term | Default | Functional role | Do not collapse into |
| --- | --- | --- | --- |
| `sati` | `remembering` | recollective awakening factor | mindfulness |
| `dhammavicaya` | `discernment of qualities` | practical discernment within the seven-member framework | analysis of doctrine |
| `viriya` | `energy` | energizing application within the framework | `sammā-vāyāma` as the default |
| `pīti` | `delight` | energizing uplift in wholesome development | `sukha`, ecstasy |
| `passaddhi` | `relaxation` | settling and easing that supports composure | vague calmness, static tranquility by default |
| `samādhi` | `mental composure` | composure member within the framework | concentration |
| `upekkhā` | `dynamic balance` | balanced steadiness without flatness or indifference | indifference, static equanimity |

## Collective Formula Guardrails

- `satta bojjhaṅgā` -> `seven awakening factors`
- `bojjhaṅgabhāvanā` -> `development of the awakening factors`
- `pañcindriyāni` / `pañca indriyāni` -> `five faculties`
- `pañca balāni` / `pañca balā` -> `five strengths`
- `indriya-bala` -> `faculties and strengths`

These collective records name frameworks.
They do not replace the governing member defaults underneath them.

## Inheritance Rules

### Awakening-Factor Side

- `bojjhaṅga` governs the framework label `awakening factor`.
- The member headword governs the first part of the derivative compound.
- A `*-sambojjhaṅga` compound should therefore normally read as:
  member default + `awakening factor`.
- Do not let a derivative member drift away from its headword default just
  because a shorter or more familiar English alternate exists in prose.
- Controlled alternates on the headword do not automatically become the default
  for the derivative compound. A narrower record must justify the change.

### Faculty Side

- `indriya` governs the framework label `faculty`.
- The member headword governs the first part of the derivative compound.
- A `*-indriya` compound should normally read as:
  member default + `faculty`.
- Keep the five-faculties family compositionally aligned unless a narrower
  compound record explicitly overrides it.

### Strength Side

- `bala` governs the framework label `strength`.
- The member headword governs the first part of the derivative compound.
- A `*-bala` compound should normally read as:
  member default + `strength`.
- Keep the five-strengths family compositionally aligned unless a narrower
  compound record explicitly overrides it.

### Faculty / Strength Interface

- `indriya` and `bala` are paired frameworks built from the same five members:
  `saddhā`, `viriya`, `sati`, `samādhi`, and `paññā`.
- The pairing does not make `faculty` and `strength` interchangeable.
- `indriya-bala` names the paired framework. It does not erase the contrast
  between governing capacity and stabilized strength.

## Keep These Distinctions Explicit

- `bojjhaṅga` is the awakening-factor framework.
  It is not just another label for faculties or strengths.
- `indriya` is governing capacity.
  It is not yet the same thing as `bala`.
- `bala` is stabilized strength.
  It is not merely a more emphatic synonym for `indriya`.
- `dhammavicaya` remains `discernment of qualities`.
  Do not collapse it into `paññā` or generic knowledge-language.
- `viriya` remains `energy`.
  Do not let the framework family drag it back into `effort` as an uncontrolled
  default.
- `passaddhi` remains `relaxation` as the governing headword default.
  Do not let awakening-factor usage drift automatically into more abstract
  tranquility-language.
- `upekkhā` remains `dynamic balance`.
  Do not flatten the awakening-factor member into indifference or static
  neutrality.

## Hindrance Interface

- This family meets the hindrance layer primarily through
  `nīvaraṇa`, `kāmacchanda`, and nearby distraction-clearing practice language.
- `nīvaraṇa` remains `distraction`, not a generic English for every obstacle.
- `kāmacchanda` remains `sensual distraction`, not wholesome motivation and not
  plain `chanda`.
- Awakening-factor development may be discussed alongside hindrance-clearing,
  but the family should not be reduced to anti-hindrance slogans.
- `sati`, `viriya`, and `samādhi` can all appear in practice-facing passages
  about hindrance-clearing without becoming synonyms for abandoning
  distractions.

## Path-Factor Interface

- `viriya` is not `sammā-vāyāma`.
  `viriya` remains `energy`, while `sammā-vāyāma` remains `right effort`.
- `sati` is not `sammā-sati`.
  `sati` remains `remembering`, while `sammā-sati` remains `right remembering`.
- `samādhi` is not `sammā-samādhi`.
  `samādhi` remains the broader mental-composure family, while
  `sammā-samādhi` remains `right mental composure` as the path factor.
- The faculty / strength family supports path language, but it does not replace
  the noble-path framework.

## Jhāna Interface

- `samādhi` and `sammā-samādhi` remain the main bridge into `jhāna` language.
- `pīti`, `passaddhi`, and `upekkhā` can all appear in jhāna-side or
  meditative-development prose, but they keep their governed headword defaults
  unless a narrower record explicitly overrides them.
- The awakening-factor family should not be flattened into jhāna-only
  terminology.
- The jhāna family should not back-govern the awakening-factor family into
  trance, absorption, or concentration language.

## Operational Reading Order

1. If the text names the seven-member development framework, start with
   `bojjhaṅga`.
2. If the text names the five-member governing-capacity framework, start with
   `indriya`.
3. If the text names the five-member stabilized-strength framework, start with
   `bala`.
4. If the text names a derivative member compound, keep the member default
   visible and add the framework label unless a narrower compound record
   explicitly overrides it.

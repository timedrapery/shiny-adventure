# Punnaji Usage Profile

## Purpose

This note records a recurring translation and interpretation profile drawn
from the following local source set by Ven. Dr. M. Punnaji Maha Thera:

- `ARIYAMAGGA BHAVANA (The Sublime Eightfold Way) - Level 1.pdf`
- `ARIYAMAGGA BHAVANA (The Sublime Eightfold Way) - Level 2.pdf`
- `ARIYAMAGGA BHAVANA (The Sublime Eightfold Way) - Level 3.pdf`
- `MN 118.pdf`
- `Return To Tranquility.pdf`

The point of this file is not to bind the repository to every source
preference. It is to preserve provenance when source-backed alternate
renderings are added to term entries. Within OSF editorial authority, this
profile is secondary to Dhammarato and Buddhadasa.

## Editorial Rule

When a Punnaji rendering differs from the current house default, the normal
first step is:

1. add the rendering as an `alternative_translation` or a source-specific
   `context_rule`
2. mention the source profile in `notes`
3. keep the current `preferred_translation` unless the repo has made an
   explicit decision to adopt the Punnaji rendering as house style

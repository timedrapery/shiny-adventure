# Punnaji Usage Profile

## Purpose

This note records a recurring translation and interpretation profile drawn
from the following local source set by Ven. Dr. M. Punnaji Maha Thera:

- `ARIYAMAGGA BHAVANA (The Sublime Eightfold Way) - Level 1.pdf`
- `ARIYAMAGGA BHAVANA (The Sublime Eightfold Way) - Level 2.pdf`
- `ARIYAMAGGA BHAVANA (The Sublime Eightfold Way) - Level 3.pdf`
- `MN 118.pdf`
- `Return To Tranquility.pdf`

The point of this file is not to bind the repository to every source
preference. It is to preserve provenance when source-backed alternate
renderings are added to term entries.

## Editorial Rule

When a Punnaji rendering differs from the current house default, the normal
first step is:

1. Add the rendering as an `alternative_translation` or a source-specific
   `context_rule`.
2. Mention the source profile in `notes`.
3. Keep the current `preferred_translation` unless the repo has made an
   explicit decision to adopt the Punnaji rendering as house style.

## Recurring Renderings

### Eightfold Path Profile

- `ariya` -> `sublime`, sometimes `supernormal`
- `magga` -> `way`
- `sammā-diṭṭhi` -> `harmonious perspective`
- `sammā-saṅkappa` -> `harmonious orientation`
- `sammā-vācā` -> `harmonious speech`
- `sammā-kammanta` -> `harmonious action`
- `sammā-ājīva` -> `harmonious lifestyle` / `harmonious life`
- `sammā-vāyāma` -> `harmonious exercise`
- `sammā-sati` -> `harmonious attention`
- `sammā-samādhi` -> `harmonious equilibrium` / `harmonious mental equilibrium`

Representative source locations:

- `ARIYAMAGGA BHAVANA - Level 2.pdf`, p. 16
- `Return To Tranquility.pdf`, pp. 28-29

### Attention / Introspection Profile

- `sati` -> `attention`
- `satipaṭṭhāna` -> `introversion of attention`, `systematic introspection`
- `ānāpānasati` -> `introspective breathing`

Representative source locations:

- `MN 118.pdf`, pp. 1-2
- `ARIYAMAGGA BHAVANA - Level 1.pdf`, pp. 51-54
- `Return To Tranquility.pdf`, pp. 17, 77, 81-82

### Tranquility / Awakening Profile

- `jhāna` -> `ecstasy`
- `samādhi` -> `mental equilibrium`, `stillness of mind`
- `passaddhi` -> `relaxation`
- `viriya` -> `strength`
- `bojjhaṅga` -> `steps to awakening` in pedagogical framing
- `upekkhā` -> `introspection` or `apperception` in some advanced practice explanations

Representative source locations:

- `ARIYAMAGGA BHAVANA - Level 2.pdf`, pp. 14, 32, 44-45
- `MN 118.pdf`, p. 3
- `Return To Tranquility.pdf`, pp. 84-88

### Cognitive / Affective Profile

- `citta` -> `mood`, `affective process`
- `mano` -> cognitive or rational side of mind
- `vedanā` -> feeling in a more affective or bodily sense

Representative source locations:

- `MN 118.pdf`, pp. 1-2
- `Return To Tranquility.pdf`, pp. 30, 42, 84

### Process / Liberation Profile

- `saṅkhārā` -> `activities`, `constructs`
- `nirodha` -> `discontinuity`, `ending`
- `bodhi` -> `awakening from the dream of existence`

Representative source locations:

- `ARIYAMAGGA BHAVANA - Level 1.pdf`, pp. 33-34
- `MN 118.pdf`, pp. 2-3
- `Return To Tranquility.pdf`, pp. 23, 41, 88

# What Is And Is Not The Path Profile

## Purpose

This document records the OSF publication *What Is And Is Not The Path* as a
house-style authority for term usage, definitions, and context rules.

This source is not being used to rename repo headwords or to replace the
repository's spelling conventions. It is being used to refine how the repo
explains and applies terms that already exist in the dataset.

## Editorial Status

This book is an OSF publication and therefore carries more authority than an
ordinary external source profile.

When its usage is clear and consistent, it can justify changes to:

- `notes`
- `context_rules`
- `definition`
- `alternative_translations`
- `preferred_translation` when that rendering is already part of OSF house
  direction

## Spelling Policy

The publication contains mixed surface forms. The repository keeps its own
canonical spellings and normalized headwords even when the book prints a
simplified or inconsistent form.

Examples:

- `Satipatthana` in the book maps to repo `satipatthana`
- `Anapanasati` in the book maps to repo `anapanasati`
- `Samadhi` in the book maps to repo `samadhi`
- `Samasankappa` in the book maps to repo `sammasankappa`
- `Mara` in the book maps to repo `mara`

The repo should absorb the book's translation policy and practical framing
without importing spelling drift from the PDF.

## High-Confidence Reinforcements

The book strongly reinforces the following existing OSF house directions:

- `dukkha` as `dissatisfaction`, with practical use of `stress`
- `sati` as `remembering`
- `samma-sati` as `right remembering`
- `samadhi` as `unification of mind`
- `sammasankappa` as `attitude`
- `anapanasati` as the central training rather than a narrow breath technique
- `kamma` as `action` or `behavior`
- `lokuttara` as supramundane without supernaturalism

## Strong Usage Themes

The book repeatedly presents:

- the path as immediately workable in daily life
- right effort as the active engine of practice
- `anapanasati` and `satipatthana` as tightly integrated rather than
  competing systems
- `jhana` in early-sutta practical terms rather than as trance doctrine
- `dhamma` as something to do, verify, and live here-now
- `dukkha -> nirodha` as a practical sequence of recognition and release

## Recommended Use In Entries

When this publication materially shapes an entry:

- mention it directly in `notes`
- use it to sharpen practical context rules
- keep repo spellings canonical even when the book prints a simpler form
- prefer stable term-level guidance over one-off rhetorical wording

# Path-Factor Supporting Terms Map

- `magga` names the whole path rather than one factor inside it.
- `ariya` keeps the family anchored as the noble path, not just a useful lifestyle list.
- `paṭipadā` keeps the practical course-of-training surface available in explanatory prose.
- `lokiya` marks wholesome but still conditioned training-side factors.
- `lokuttara` marks the beyond-the-world side of the same path family.
- `sammā-ñāṇa` and `sammāvimutti` preserve the tenfold completion close of MN 117.
- `ariya-atthangika-magga` remains the collective label for the eightfold whole and should point readers back to the governed factor records.

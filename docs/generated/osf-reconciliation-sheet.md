# OSF Reconciliation Sheet

| Term | Repo default | OSF-facing usage | Classification | Governance decision |
| --- | --- | --- | --- | --- |
| suññatā | emptiness | OSF-facing materials support emptiness, while Buddhadasa-line prose also uses voidness and void-mind language. | ALIGN | Use emptiness as the default rendering. Allow voidness as a controlled alternate only when the context clearly supports it. Do not let this drift into nihilist or mystical language. |
| animitta | signless | OSF-facing materials do not strongly displace the default, while practical explanation can unpack the term as without sign or not taking up signs. | TOLERATE ALTERNATE | Use signless as the default rendering. Use without-sign language in explanatory prose only. Do not let this drift into formlessness or mystical vagueness. |
| appaṇihita | wishless | Practical explanation can unpack the term as without placing desire or as narrower desireless language, but the family needs a sharper default. | TOLERATE ALTERNATE | Use wishless as the default rendering. Allow explanatory alternates only when the context clearly supports them. Do not let this drift into passivity, apathy, or no-goals language. |
| nibbāna | nibbāna | OSF and Buddhadasa-facing materials strongly use cooling, coolness, and cooling down in explanation. | TOLERATE ALTERNATE | Keep nibbāna untranslated by default. Allow cooling-language in OSF-facing explanatory prose only. Do not let this drift into mystical or absolutized language. |
| nirodha | quenching | OSF and Dhammarato materials often use stopping, ending, or cessation in practical explanation. | TOLERATE ALTERNATE | Use quenching as the default rendering. Allow ending or cessation only in controlled OSF-facing explanation. Do not let this drift into loose generic cessation language. |
| sati | remembering | Dhammarato and OSF path-language strongly support remembering, while some rough glossary or coaching prose uses awareness-language. | ALIGN | Use remembering as the default rendering. Allow awareness only as a controlled explanatory alternate. Do not let this drift into mindfulness or bare-attention language. |
| vimutti | release | OSF-facing prose can use liberation, but the repository default is sharper as release. | TOLERATE ALTERNATE | Use release as the default rendering. Allow liberation as a controlled OSF-facing alternate only when the context clearly supports it. Do not let this drift into generic spiritual-freedom rhetoric. |
| ānāpānasati | ānāpānasati | OSF glossary explains it as remembering to look at in-and-out breathing. | TOLERATE ALTERNATE | Keep the untranslated default. Allow the OSF gloss in explanatory prose only. Do not let this collapse into generic breath-meditation language. |
| samādhi | mental composure | Dhammarato and OSF materials strongly reject concentration and support a practical composure or collectedness handling. | ALIGN | Use mental composure as the default rendering. Do not let this drift into concentration language. |
| citta | feeling mind | OSF explanatory material can use mind, heart-mind, or center of consciousness, especially in practice prose. | TOLERATE ALTERNATE | Use feeling mind as the default rendering. Allow OSF-facing alternates in explanatory prose only. Do not let this collapse into cognition or consciousness language. |
| passaddhi | relaxation | Rough peace- or calm-language could easily flatten passaddhi into santi or generic tranquility. | REFUSE DRIFT | Use relaxation as the default rendering. Do not let this drift into undifferentiated peace-language. |

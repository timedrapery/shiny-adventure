# Path-Factor Tenfold Sequence Sheet

## Training Factors

- `sammā-diṭṭhi` -> `right view`
- `sammāsaṅkappa` -> `right attitude`
- `sammā-vācā` -> `right speech`
- `sammā-kammanta` -> `right action`
- `sammā-ājīva` -> `right livelihood`
- `sammā-vāyāma` -> `right effort`
- `sammā-sati` -> `right remembering`
- `sammā-samādhi` -> `right mental composure`

## Completion Sequence

- `sammā-samādhi` stays the eighth path factor and the governed opening into the completion-side close.
- `sammā-ñāṇa` -> `right knowledge`
- `sammāvimutti` -> `right release`
- Keep explicit that the trainee is eight-factored while the one beyond training closes the sequence with right knowledge and right release.

## Family Contrasts

- `lokiya` -> `worldly`
- `lokuttara` -> `beyond-the-world`
- `paṭipadā` keeps the practical course language available without replacing `magga`.

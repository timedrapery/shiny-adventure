# Jhana Path Brief

## Governing Rule

- `sammﺅ·-samﺅ·dhi` is the first, second, third, and fourth `jhﺅ·na`.
- Keep `right unification of mind` as the default rendering for the path factor while making that fourfold identity explicit in notes and explanations.

## Core Defaults

- `jhﺅ·na` -> `mental theme`
- `jhﺅ·yati` -> `think like this`
- `samﺅ·dhi` -> `unification of mind`
- `vitakka` and `vicﺅ·ra` stay coordinated as the practical pair in the first mental theme.

## Drift Guards

- Do not widen ariya samma samadhi into a vague concentration category.
- Do not let jhana drift into trance, absorption, or ecstasy language.
- Keep the hindrance-clearing and practical-development frame visible across the cluster.
- Keep the fourfold sequence stable unless a controlled technical reason requires untranslated Pali.

# Dukkha-Nirodha Scope Sheet

- `ariyasacca` keeps the four truths ordered by the repository's `dukkha / dukkha-nirodha` scope rule.
- `dukkha` remains the first governing side of that scope as `dissatisfaction`.
- `nirodha` remains the ending side of that scope as `quenching`.
- `magga` and `paṭipadā` stay readable inside that scope rather than drifting into generic self-improvement language.
- `samudaya` stays the origin-side explanatory truth and should remain readable as what the present pattern is heading into, not as abstract creation-talk.

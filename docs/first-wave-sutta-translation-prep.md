## Sutta Prep

### `SN 12.2` Paṭiccasamuppāda-vibhaṅga Sutta

- Review first: dependent-arising headwords and link terms, especially
  `paṭiccasamuppāda`, `paccaya`, `nirodha`, `jāti`, `phassa`, `vedanā`,
  `taṇhā`, `upādāna`, `nāmarūpa`, and `viññāṇa`.
- Consult:
  [generated/dependent-arising-formula-sheet.md](generated/dependent-arising-formula-sheet.md)
  and
  [generated/dependent-arising-translator-brief.md](generated/dependent-arising-translator-brief.md).
- Watch for: `condition` drifting into `cause`, `birth` drifting into
  `rebirth` without local need, and the `phassa -> vedanā -> taṇhā -> upādāna`
  pivot losing its governed practical force.
- Likely update targets after translation: only the link formulas or local
  notes that prove unstable in repeated use.

### `DN 2` Sāmaññaphala Sutta

  `vicāra`, `pīti`, `sukha`, `nīvaraṇa`, and `saṃvara`.
- Consult:
  [first-jhana-entry-sheet.md](first-jhana-entry-sheet.md),
  [generated/jhana-formula-sheet.md](generated/jhana-formula-sheet.md),
  [generated/jhana-sequence-sheet.md](generated/jhana-sequence-sheet.md), and
  [generated/jhana-path-brief.md](generated/jhana-path-brief.md).
- Watch for: the hindrance-clearing lead-in, the first-jhāna pair language,
  `samādhi` versus the formula-specific `cetaso ekodibhāva`, and long
  practice-progression prose slipping into mixed English registers.
- Likely update targets after translation: only the jhāna formula minors or
  hindrance-side notes that still feel unstable after direct use.

  [generated/dependent-arising-translator-brief.md](generated/dependent-arising-translator-brief.md),

## Use Rule

Before drafting any one of these five suttas:

1. review the named headwords, compounds, and formula records together
2. read the listed control sheets before opening a translation file
3. add new records only when repeated use or real drift pressure justifies them
4. update live policy first, then refresh any affected generated surfaces

# Consummation / Unconditioned Interface Map

This note governs the repository's handling of the
`nibbāna / amata / asaṅkhata / santi / nibbuta / parinibbāna` interface.
It is not a general essay. It exists to stop consummation-side language from
collapsing into one vague English blur.

## Core Distinctions

| Term | Default | Functional Role | Do Not Collapse Into |
| --- | --- | --- | --- |
| `nibbāna` | `nibbāna` | Governing consummation-side headword | generic peace, generic cessation, decorative `nirvana` |
| `amata` | `deathless` | Deathless designation or facet | immortality, eternal self |
| `asaṅkhata` | `unconditioned` | Unconditioned designation or contrast term | vague metaphysics, casual synonym for `nibbāna` |
| `santi` | `peace` | Peace-side designation | mood-level calm, `passaddhi` |
| `nibbuta` | `cooled` | Adjectival cooling/quenching condition | `nibbāna`, generic peace |
| `parinibbāna` | `final nibbāna` | Final-completion frame | death, annihilation |

## Interface With Already-Stabilized Families

| Family | Default | Interface Role |
| --- | --- | --- |
| `vimutti` | `release` | Result-side release language pointing toward consummation |
| `mutti` | `freedom` | Shorter result noun distinct from the consummation designations |
| `nissaraṇa` | `escape` | Way-out language rather than consummation naming |
| `tarati` / `tiṇṇa` | `crosses over` / `crossed over` | Crossing imagery rather than consummation naming |
| `yogakkhema` | `security from the yoke` | Stabilized security condition linked to the yoke family |
| `virāga` | `fading` | Process-side cooling/fading language |
| `nirodha` | `quenching` | Culmination-side quenching language |
| `khaya` | `wearing away` | Exhaustion/wearing-away language |
| `upasama` | `calming` | Quieting language that can point toward peace without replacing it |
| `passaddhi` | `relaxation` | Practice-sequence relaxation, not consummation peace |

## Operational Reading Order

1. If the text names the governing consummation-side destination, prefer `nibbāna`.
2. If the text highlights deathlessness, prefer `amata`.
3. If the text highlights the conditioned/unconditioned contrast, prefer `asaṅkhata`.
4. If the text highlights peace as designation, prefer `santi`.
5. If the text uses adjectival cooling or quenching language, prefer `nibbuta`.
6. If the text names the final-completion frame, prefer `parinibbāna`.

## Process, Result, and Consummation

| Process / Result Side | Consummation Side |
| --- | --- |
| `virāga` | `nibbuta` |
| `nirodha` | `nibbāna` |
| `vimutti` | `amata` / `asaṅkhata` / `santi` |
| `tarati` / `tiṇṇa` | `yogakkhema` |

This is a translator-facing control aid, not a rigid universal sequence.
The texts can bring these terms close together without making them
interchangeable.

## Translator Guardrails

- Do not translate all consummation-side terms with `peace`.
- Do not translate all consummation-side terms with `the unconditioned`.
- Do not let `amata` drift into immortality or eternal-self language.
- Do not let `asaṅkhata` drift into abstract metaphysical prose.
- Do not let `santi` drift down into mood-level calm or practice-level relaxation.
- Do not let `nibbuta` become a casual synonym for `nibbāna`.
- Do not let `parinibbāna` become generic death-language.

## Imagery and Designation Discipline

- Keep `nibbāna` untranslated by default in source-facing doctrinal prose.
- Use `quenching` or `coolness` as controlled glosses, not as automatic replacements.
- Let `amata` keep its deathless force.
- Let `asaṅkhata` keep its conditioned/unconditioned contrast.
- Let `santi` keep peace-language without sentimentality.
- Let `nibbuta` keep adjectival cooling force rather than turning it into a noun.

## Contrast Matrix

| If the text suggests... | Prefer | Avoid by Default |
| --- | --- | --- |
| governing consummation headword | `nibbāna` | generic `peace`, generic `liberation` |
| deathless side of consummation | `deathless` | `immortal`, eternal-life language |
| conditioned/unconditioned contrast | `unconditioned` | vague transcendence-language |
| peace as designation | `peace` | mood-level calm, `relaxation` |
| cooled or quenched as adjective | `cooled` | `nibbāna`, `at peace` |
| final-completion nibbāna frame | `final nibbāna` | `death`, `annihilation` |

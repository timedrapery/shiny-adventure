# SN 56.11 Translation Notes

This document is the companion rationale for
[SN 56.11: Dhammacakkappavattana Sutta](sn56-11-dhammacakkappavattana-sutta.md).
The main translation is the primary study surface; this file records debated
translation choices, source-audit calls, and edition notes that govern it.

## Source Base

- Primary source: the Pali text of SN 56.11 as segmented in SuttaCentral's
  Bilara data.
- Control surface: the repository's current term policy, especially
  `ariyasacca`, `dukkha`, `dukkha-ariyasacca`, `samudaya`,
  `samudaya-ariyasacca`, `nirodha`, `nirodha-ariyasacca`, `magga`,
  `magga-ariyasacca`, `patipada`, `tanha`, `ya-tanha-ponobbhavika`,
  `nandiraga-sahagata`, `tatratatrabhinandini`, `mn141-tanha-release-tail`,
  `pancupadanakkhandha`, `parinna`, `pahana`, `bhavana`, `nana`, `panna`,
  `vijja`, `vimutti`, and the generated four-noble-truths sheets.
- Working method: the two-extremes opening, the four truth definitions, the
  three-turning task sequence, and the closing Kondañña / wheel-setting
  material are governed together so SN 56.11 functions as one translation
  control passage rather than as a set of familiar slogans.

## Governing Decisions

- `Dhammacakkappavattana` is rendered `Setting the Wheel of Dhamma Rolling`.
- `ariyasacca` remains `noble truth`.
- `dukkha` remains `dissatisfaction`.
- `samudaya` remains `origin`.
- `nirodha` remains `quenching`.
- `magga` remains `path`, while `paṭipadā` remains `path of practice`.
- The second-truth definition line stays segmented as `that ignorant wanting
  that leads to renewed becoming`, `accompanied by relishing and passion`, and
  `delighting here and there`.
- The third-truth tail reuses the governed release sequence `giving it up,
  relinquishing it, freedom, non-attachment`.
- The repeated truth-task lines keep `should be fully understood`,
  `should be abandoned`, `should be directly known`, and `should be developed`
  as the working task verbs.
- The cognition cluster in the turning sequence stays explicit as `vision`,
  `knowledge`, `discernment`, `clear knowledge`, and `light`.
- Kondañña's insight is rendered `Whatever is subject to origin is all subject
  to quenching` so the closing line stays aligned with the repo's current
  `samudaya` and `nirodha` defaults.

## Re-audit Calls

- The two-extremes opening avoids inherited sermon cadence. The point is not
  merely to denounce indulgence and pain, but to set up the practical middle
  path as the governed alternative.
- `majjhimā paṭipadā` is kept as `middle path of practice`, not collapsed into
  bare `middle path`, so `paṭipadā` remains distinct from `magga`.
- The first truth keeps `five clung-to heaps` rather than reopening aggregate
  vocabulary locally.
- The second-truth definition line keeps `renewed becoming` instead of
  rebirth-shorthand so `ponobbhavikā` does not narrow prematurely.
- `nandī` and `rāga` remain distinct in the coordinated middle segment
  `relishing and passion`.
- The third-truth tail stays coordinated rather than being compressed into one
  abstract release word.
- The turning sequence is written out in full because the discourse's name
  depends on the wheel-like repetition of four truths across three tasks.
- `ti-parivaṭṭa` / `dvādasākāra` is handled as `three-round, twelve-aspect`
  knowledge and seeing to stay readable without scholastic ornament.
- The wheel-setting close uses `set rolling` / `cannot be rolled back` so the
  image remains forceful and usable in plain English.

## Practice Clarifications

- This edition treats SN 56.11 as a practical control passage, not as a
  ceremonial origin-story detached from training.
- The first half defines the middle path and the four truths; the second half
  turns them into tasks and completed tasks.
- The truth-task sequence matters because it prevents the four truths from
  remaining static doctrinal labels.
- Kondañña's Dhamma-eye line is handled as a concise origin-and-quenching
  insight, not as an abstract metaphysical slogan.
- The final wheel-setting scene matters because it identifies the discourse as
  the release of a teaching surface that cannot simply be rolled back into
  inherited drift.

## Editorial Presentation

- The main translation file uses section headings so the discourse can function
  as a study text.
- Those headings are editorial aids, not claims that such headings stand in
  the root text itself.
- The four truth definitions are laid out explicitly because this sutta is now
  a truth-family control surface.
- The three-turning sequence is also laid out explicitly because the wheel
  image and the task logic both depend on that repetition being visible.
